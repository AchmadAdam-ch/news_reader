# Aplikasi Pembaca Berita

Aplikasi Flutter untuk membaca berita terkini dari Indonesia dan seluruh dunia.

## Fitur
- Berita Utama & Pencarian berita
- Penyaringan kategori
- Tandai berita favorit
- Bagikan berita (Bonus)
- Mode Gelap (Bonus)
- Pengujian Unit

## Petunjuk Pengaturan
1. Pastikan Flutter sudah terinstal: `flutter doctor`
2. Kloning repositori dan masuk ke direktori.

3. Dapatkan dependensi: `flutter pub get`
4. Jalankan aplikasi di Chrome (nonaktifkan keamanan web untuk akses API):

```bash
flutter run -d chrome --web-browser-flag "--disable-web-security"
